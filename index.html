The canvas 3D transition and animation library taccGL has been installed. It allows you to animate CSS formatted HTML elements and 3D objects on your web site using HTML 5 3D canvas. Please click on the example links below to run the examples. If they work fine and colored logos above appear the library is probably installed successfully.

It is not intended to install this taccgl-package directly on a public web server since it contains some documentation and example files. Follow the instructions Installing the library on https://www.taccgl.org for installation on a public site.

If you are using the library locally without web server (i.e. with "file://" URLs) you need to set some options for additional examples and your experiments to work, see local usage.

NOTE This is BETA Software and known to contain bugs. It is not implemented completely and will probably not run on all browsers and client computers.

The library runs best on a client/browser with decent performance on HTML 5 canvas. If the performance is not sufficient features are silently disabled (e.g. the display of shadows). The idea is to keep the underlying HTML pages operational, if necessary with simpler or even disabled taccGL ™ animations. See Restrictions on https://www.taccgl.org/ for details.
